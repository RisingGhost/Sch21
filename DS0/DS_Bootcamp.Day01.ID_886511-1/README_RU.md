1. Напишите программу, которая будет кодировать или декодировать любую строку, используя заданный сдвиг, согласно переданному аргументу:

       $ python3 caesar.py encode 'ssh -i private.key user@school21.ru' 12
       eet -u bduhmfq.wqk geqd@eotaax21.dg
       $ python3 caesar.py decode 'eet -u bduhmfq.wqk geqd@eotaax21.dg' 12
       ssh -i private.key user@school21.ru

2. Если скриптам передана строка, содержащая кириллические символы, они должны выбросить исключение: «Скрипт пока не поддерживает ваш язык». Если передано неверное количество аргументов, выбросьте исключение.
